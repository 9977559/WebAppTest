Comp.5210 Web Application Implementation

The aim of this course is to provide students with the skills to select and use appropriate development methodologies, develop user interfaces using software development standards, create scripted components to given specification, and install and configure frameworks and applications.

  • Select an appropriate software development paradigm.
  • Use software development standards to implement a user interface.
  1. Install and configure a web-based solution using web scripting.
  2. Initialise system data.
